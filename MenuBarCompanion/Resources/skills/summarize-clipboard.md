# Summarize Clipboard

Summarize whatever text is currently on the user's clipboard. Provide a concise, clear summary.

Read the clipboard contents and produce a summary that captures the key points. Adjust your summary style based on the content:

- For articles/blog posts: Key arguments, conclusions, and notable quotes
- For code: What the code does, its purpose, and any notable patterns
- For emails/messages: The main ask or information, action items
- For data/tables: Trends, outliers, key statistics

Keep the summary concise but complete. Prioritize actionable information.

{extra_instructions}

## When to Use This Skill

- User asks you to summarize their clipboard
- User copies something and wants a quick overview
